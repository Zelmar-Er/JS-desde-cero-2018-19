Spread Operator

U operador de expansión. Su sintaxis es: ...
const numbers = [-12,2,3,23,43,2,3];
console.log(numbers) // (7) [-12, 2, 3, 23, 43, 2, 3]
console.log(...numbers) // -12 2 3 23 43 2 3
 Al utilizarlo obtenemos los valores por separado.
Esto nos permite, enviar elementos de un array a una función. Ej:
const addNumbers = (a,b,c) => {
console.log(a+b+c)
}
let numbersToAdd = [1,2,3]
console.log(addNumbers(1,2,3)) // 6
console.log(addNumbers(numbersToAdd)) // 1,2,3undefinedundefined
console.log(addNumbers(...numbersToAdd)) // 6
 Añadir un array a otro. Ej
let users = ['pablo','javier','alejo'];
let newUsers = ['valentina','rosa','juana'];
console.log(users); // (3) ['pablo', 'javier', 'alejo']
users.unshift(newUsers);
console.log(users); // (6) ['valentina', 'rosa', 'juana', 'pablo', 'javier', 'alejo']
 Copiar arrays.
let numbersTwo = [...numbers]
console.log(numbersTwo) // (7) [-12, 2, 3, 23, 43, 2, 3]
Concatenar arrays.
let numOne = [1,2,3];
let numTwo = [4,5,6];
let numThree = [...numOne,...numTwo];
 console.log(numThree) // (6) [1,2,3,4,5,6]
Enviar un número indefinido de argumentos a una función. Parámetros REST
const restParams = (...x) =>{
console.log(x)
}
restParams(2,3,5,2,65,'a','g','r',7,'h') // (10) [2, 3, 5, 2, 65, 'a', 'g', 'r', 7, 'h']
 Librería math.
Se puede utilizar con las funciones de la librería.
consle.log(Math.max(...numbers)) // 43
 Eliminar elementos duplicados.
console.log(new Set(numbers)) // -12,2,3,23,43 (nos devuelve un objeto set sin duplicados)
console.log([...new Set(numbers)]) // [-12,2,3,23,43] (nos devuelve un array sin duplicados)
