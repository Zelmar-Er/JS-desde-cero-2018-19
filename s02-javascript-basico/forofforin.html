Bucles for of y for in

Simplifican el bucle for tradicional sin tener que darle un número de vueltas ni realizar un incremento o decremento.
let names = ["Paco","José","Paula","María"];
for(let name of names){
console.log(name);
}
 1° vuelta: Paco
2° vuelta: José
3° vuelta: Paula
4° vuelta: María
for(let name in names){
console.log(name);
}
 1° vuelta: 0
2° vuelta: 1
3° vuelta: 2
4° vuelta: 3

Palabras reservadas de bucles:
break - Rompe el bucle - Se salta la(s) posicion(es) que el indiquemos y despúes continúa su ejecución.
continue - Salta de posición - Salta la(s) posicion(es) que le indiquemos y continúa con el bucle.
