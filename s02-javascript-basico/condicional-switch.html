Estructura Switch

Se utliza para elejir un camino de varios preestablecidos. Tenemos 2 tipos principales:

Sintaxis simple:
switch(evaluación){
case n1:
//código
break;
case n2:
//código
break;
default:
//código }

Sintaxis múltiple:
switch(evaluación){
case n1:
case n2:
//código
break;
default:
//código }
