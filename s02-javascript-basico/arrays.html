Arrays I y II

Arrais I

Son estructuras que nos permiten almacenar varios datos y agruparlos.
Se pueden llenar con cualquier tipo de dato válido en JavaScript y deben ir separados por comas.
Se pueden mezclar tipos de datos, pero no es recomendable.
Se declaran con llaves cuadradas o corchetes[].
Pueden declararce vacíos o con contenido ya establecido.
Pueden añadirce o eliminarse elementos en el momento que queramos.
let numero = 5;
lat array = [];
let numeros [1,2,3,4,5];
Cada uno de los elementos podrá ser identificado por su índice, es decir su posición.
Los índices empiezan a contar desde 0.

Arrais II

Propiedas:

.length - devuelve el número de posiciones que contiene el array.
Métodos:
Array.isArray() - devuelve true si la variable es un array.
Eliminar un elemento:
.shift() - Elimina el primer elemento del array y devuelve ese elemento.
.pop() - Elimina el último elemento de un array y devuelve ese elemento.
Añadir elementos:
.push(e1,e2,e3,...) - añade uno o más elementos al final del array y devuelve la nueva longitud.
.unshift(e1,e2,e3,...) - añade uno o más elementos al comienzo del array y devuelve la nueva longitud.
Buscar elementos:
.indexOf() - Devuelve el primer índice del elemento que coincida con el valor especificado, o -1 si ninguno es encontrado.
lastIndexOf() - Devuelve el último índice del elemento que coincida con el valor especificado, o -1 si ninguno en encontrado.
.reverse() - invierte el orden de los elementos del array.
.join('separador') - devuelve un string con el separador que indiquemos, por defecto son comas.
.splice(a,b,item) - cambia el contenido de un array eliminando y/o agregando nuevos elementos.
a - Indice de inicio. Si solo se indica el indice, borrara los elementos despues de este.
b - Número de elementos (opcional). Si b es distinto a 0 elimina el número de valores que indiquemos apartir de a.
items - Elementos a añadir en el caso de que se añadan (opcional).
Ej: let menosAnimales = animales.splice(3);
let algunosAnimales = animales.splice(5,2);
let nuevosAnimales = animeles.splice(2,0,'coballo','perro','conejo');
.slice(a,b) - Extrae elementos de un array desde el índice a hasta el indice b. Si no existe b lo hace desde a hasta el final, si no existe ni a ni b hace una copia del original.
