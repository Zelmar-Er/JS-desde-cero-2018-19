Operador Ternario

Se utiliza cuando una condición será true o false, al igul que el if.
Su ejecución puede tener una o varias sentencias, en este caso irán separadas por comas y entre paréntesis.
Ejemplo:

Caso 1: (condición) ? true : false;
Caso 2: (condición) ?
(primera sentencia, segunda sentencia) :
(primera sentencia, segunda sentencia);
