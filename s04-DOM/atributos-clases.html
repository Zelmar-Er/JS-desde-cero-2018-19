Atributos y clases

Atributos:

element.getAttribute('attribute') - para obtener un atributo.
element.setAttribute('attribute','value') - para agregar un atributo y su valor.
Clases:
element.classList.add('class','class',...) - para agregar clases.
element.classList.remove('class','class',...) - para remover clases.
element.classList.contains('class') - para saber si el elmento contiene dicha clase.
element.classList.replace('oldClass','newClass') - para reemplazar una clase por otra.
