Objetos Nativos y Timers

Objeto window - Es el objeto global, de él descienden todos los objetos

alert()
prompt()
confirm()

Objeto console - Es el objeto que contiene la consola del navegador API Consola

console.log() console.dir() // Similar a .log() permite desplegar cuando .log() no lo hace
console.error() // Nos permite sacar mensajes de error
console.table() // Nos muestra la información en forma de tabla

Objeto location - Es el objeto que contiene la barra de direcciones API Location

location.href // Devuelve la dirección actual, también sirve para cambiar la pagína
location.protocol // Devuelve el protocolo de la pagína (http || https || ftp)
location.host // Da la dirección del dominio principal
location.pathname // Da los subdominios donde estamos metidos
location.hash // Es una forma de pasar parametros entre pagínas, se puede utilizar para tener un solo HTML y en función del hash cargar la información
location.reload() // Recarga la pagína

Objeto history - Trabaja con el historial de la pestaña con la que estamos trabajando

back() // Va hacia atras en el historial
forward() // Va hacia adelante en el historial
go(n|-n) // Nos permite navegar, en el historial de navegación actual, x pagínas hacia adelante o atras
length // Nos muestra la cantidad de pagínas en el historial

Objeto date - Este objeto permite tener información datos del tiempo, que van desde los segundos hasta años. Los objetos date necesitar ser instanciados para poder utilizarlos.
Ejemplo: const date = new Date();
console.log(date.getDay()) // Da el número del día, empezando desde 0 al 6 sistema anglosajon

Timers Timeout: setTimeout(()=>{code}, delay-in-miliseconds)
- Hace que se ejecute la función despues de delay. Si lo referenciamos mediante una variable/constante podemos pararlo con clearTimeout(referencia) Interval: setInterval(()=>{code}, delay-in-miliseconds)
- Hace que se ejecute la función cada delay milisegundos. Si lo referenciamos mediante una variable/constante podemos pararlo con clearInterval(referencia)
