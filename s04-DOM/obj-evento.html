Objeto Evento

El objeto evento es el parametro pasado en la función de un event listener, ejemplo
addEventListener('click',(evento)=>{
console.log(evento);
}
 Este parametro suele llamarse 'e' o 'event' y nos pasara información sobre el evento.

Los eventos se escuchan con .addEventListener(), pero tambien se pueden forsar utilizando el evento como si fuera un metodo como por ejemplo .click(), que forsaria el evento click.
